
# פרויקט מסכם - רשתות תקשורת

ברוכים הבאים לרפוזיטורי של פרויקט הגמר שלנו. כאן נמצאים כל קבצי הקוד והנתונים.

###  איפה ההסברים?

תיעוד מלא של הפרויקט, כולל הוראות הרצה מפורטות, ניתוח תעבורה ב-Wireshark, צילומי מסך ומסקנות, נמצאים בקובץ ה-Word המצורף בתיקייה זו:
 **`עבודה רשתות תקשורת.docx`**

### קבצים עיקריים

* **server.py** – קוד השרת (תומך בריבוי משתמשים).
* **client.py** – קוד הלקוח.
* **Jupyter Notebook** – מחברת הקוד לחלק הראשון (יצירת פאקטים).
* **csv file** – קובץ הנתונים ששימש לסימולציה.
